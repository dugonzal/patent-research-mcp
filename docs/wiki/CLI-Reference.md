# CLI Reference

| Command | Description |
|---------|-------------|
| `patent-research seeds` | List seed patents |
| `patent-research fetch <num>` | Download patent |
| `patent-research sections <num>` | Extract sections |
| `patent-research patterns` | List patterns |
| `patent-research export` | Generate summary |

## Environment

| Variable | Default | Description |
|----------|---------|-------------|
| `PATENT_RESEARCH_DATA` | `$CWD/data` | Data storage path |
| `RESEARCH_PLUGIN` | — | Plugin directory path |
