# MCP Tools Reference

| Tool | Description |
|------|-------------|
| `patent_seed_list` | List seed patents with metadata |
| `prompt_get` | Get prompt templates (core + plugin) |
| `patent_fetch` | Download patent HTML, TXT, PDF |
| `patent_get_sections` | Extract title, abstract, claims, description |
| `architecture_card_save` | Save architecture analysis |
| `claims_firewall_save` | Save claims risk assessment |
| `pattern_save` | Save reusable pattern |
| `pattern_list` | List saved patterns |
| `pattern_compare` | Compare patterns across patents |
| `research_export_markdown` | Generate research summary |
| `suggested_module_proposal` | Generate module proposal template |
