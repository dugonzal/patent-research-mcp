# CLI Reference

| Command | Description |
|---------|-------------|
| seeds | List seed patents |
| fetch <num> | Download patent |
| sections <num> | Extract sections |
| patterns | List patterns |
| export | Generate summary |

## Environment
| Variable | Default | Description |
|----------|---------|-------------|
| PATENT_RESEARCH_DATA | \$CWD/data | Data path |
| RESEARCH_PLUGIN | — | Plugin path |
